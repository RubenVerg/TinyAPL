---
glyph: '⍒'
pattern: 'r←x⍒y'
name: Sort By Down
---

`r` is the major cells of `x` sorted according to the descending grade of `y`: `r←x⊇⍨⍒y`. Supports the `tolerance` extra argument for the comparisons; supports the `backward` extra argument, which returns the&nbsp;<primitive key="sort_by_up" />&nbsp;of `y`.
